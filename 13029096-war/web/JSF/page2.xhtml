Accessing Managed bean

JSF provides Expression Language (EL) to access and modify underlying data in a managed bean from a JSF page as well as invoking public methods.

Accessing Properties

Using EL a bean property can be access as follows;

\#{mybean.value}

In the above expression EL relies on setter and getter method of the 'value' property of mybean Java bean. EL implicitly calls getValue() for displaying it's value and setValue() to set given input value from html form input.

Accessing Methods

Similarly, using EL a public method in a managed bean can be invoked as follows;

\#{mybean.processPage()}

And arguments can be passed as below;

\#{mybean.processValue(value)}
